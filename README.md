## 🧑‍💻 About Me

Java and Android Developer with a unique and inspiring journey from tofu seller back to professional developer and educator. Passionate about creating meaningful applications and sharing knowledge with the next generation of developers.

After starting my journey in programming in 2017, I worked as a JavaFX developer before exploring other technologies like C# and .NET. In 2019, I took a break to return to my hometown and run a small business selling fried tofu. During this time, I continued learning and discovered Android Development, creating applications to help my local community. Since 2024, I've transitioned back to technology full-time, now teaching Java and Android development to young enthusiasts online.

This journey has given me a unique perspective on both the technical and human aspects of software development, making me a more empathetic developer and educator.

- 🔭 Currently working on [Pakpi Calendar](http://play.google.com/store/apps/details?id=it.saimao.pakpicalendar)
- 🌱 Learning Java Spring and Spring Boot to expand backend development skills
- 👯 Open to collaborating on Spring Projects and Android applications
- 📚 Teaching Java and Android development to enthusiasts online
- 🌍 Based in Myanmar (Burma)
